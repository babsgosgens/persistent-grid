Introduction

Persistent Grid is a modular CSS framework built with SASS and Compass. It basically comes in three flavours: Grid, Base and Framework.

Grid

Grid is the most basic execution of Persistant Grid. If you are only interested in the layout system then this is your cup of tea.

Base

Basic provides the grid and a set of base styles to build your project on. It includes a Reset module, a Type module and a bunch of (very useful) base styles. If you're looking for a solid base to start from, then this is probably your flavour of choice.

Framework

Framework1) is an extension to Base. It provides a collection of commonly used UI elements like buttons, forms, tabs and a whole lot more. And you don't have to stick with what's there either. You can add as well as override any of the framework modules. More information is to come…

1) Framework is still under heavy construction.

Installing Persistent Grid (pre-alpha)

During the development of Persistent Grid (or at least until a first release candidate) you can install Persistent Grid as an Ad-Hoc Compass extension. Ad-Hoc extensions reside in a directory named extensions in the root of your Compass project.

Persistent Grid will be available as a gem as soon as it's ready to ship.

How to add Persistent Grid to your Compass project

Create a directory named extensions in the root of your Compass project directory.

Unless of course directory does already exist.

Make sure to replace "path_to_your_compass_directory" with the actual path to your Compass directory.

b) Install using an archive

You can also add Persistent Grid by extracting an archived copy of Persistent Grid into the extensions directory. Rename the extracted directory from persistentgrid-master to persistentgrid.
