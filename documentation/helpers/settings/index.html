Presets

The presets are to help you build your site more quickly. Like module settings, these can be overridden or extended by your project settings. The settings are grouped:

Breakpoints

Breakpoints help you implement consistent media queries. This is still under heavy construction and I am considering switching to Breakpoints instead as they’re approach is much like what I had in mind (http://breakpoint-sass.com/).

Colours

All 140 color names as per the HTML and CSS color specification (http://www.w3schools.com/html/html_colornames.asp).

Font Sizes

Font sizes implement consistent font sizes accross a project. Currently, font-sizes are under construction.

Fonts

This is where Persistent Grid’s Type Lists are defined. A Type List is a list of type properties: style, variation, weight, family. They help you implement “real webfonts”. Currently, these type presets are available. Use the example preset to define your own additional Type Lists.

The Typography module has mixins and methods to work with Type Lists in your projects.
