About Persistent Grid

Persistent Grid is a configurable grid system that is incredibly flexible yet super simple to use. Compose any layout as Persistent Grid is not fixed to any number of columns. Responsive webdesign has never been this easy.

CSS to taste

By default, Persistent grid is configured to render style rules for whole, half, third and fourth columns. With this simple preset, you can already compose tens of different column layouts. If set (and it is by default), columns-widths are negated on a certain breakpoint (which of course is configurable) unless a column-width is persistent.

Mix flavours, in any combination.

Persistent Grid is more than just the sweetest grid on the block. It is also a reset, a set of base styles and a collection of commonly used UI elements: buttons, lists, forms an much more.

Create new modules, it’s super easy.

With extensibility in mind, a great amount of thought has been put into the organization of directory structure.

Persistent Grid is a modular CSS framework built with SASS and Compass. It comes in three flavours: Grid, Base and Extended.

Grid

Grid is the most basic execution of Persistant Grid. If all you need is a good layout system to complete your project then this is your flavour of choice.

compass install persistentgrid/grid

Base

Basic provides a little extra to the grid: a set of base styles to build your project on. It includes the Grid, Reset, Type module and Base Styles modules. It provides a solid base to build your stylesheet on.

compass install persistentgrid/base

Modules

Modules1) adds some of commonly used UI elements to the mix. It's an ever growing colelction of buttons, forms, tabs and a whole lot more. And you don't have to stick with the core modules. In fact you can add or override any of the provided modules. More information is to come…

compass install persistentgrid/modules

1) Modules is still under heavy construction.
