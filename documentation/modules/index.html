Skin module

The skin module is where all your project's settings reside. Any module settings are imported through the Skin modules and project specific settings like colours are set here.

Helper module

The Helper module is a collection of helper functions and mixins.

Base modules

Base modules are the core of each project. Base modules are responsible for your site's soul. They define your Type and whitespace and set the base for all modules.

Reset

Based on Compass Reset - which is in turn based on Eric Meyer's reset - I have assembled a custom reset that is a little more configurable and specifically adapted for Persistent Grid.

Type

Type declares the project's default typeface and style, font size and leading.

Vertical Grid

Based on Compass Vertical Rhythm. The base-line-height and base-font-size are inherited from the Type-module. If the baseline was not yet set in the Type module, or the Type module is not included in your project, you can set the baseline here.

Base Styles

Next up are the base styles. Base styles are usually applied to element selectors and define the default behavior for any element throughout your project. Base styles can be overruled by modules and skins. See: http://smacss.com/book/type-base for more information.

Persistent Grid

Persistent Grid does not define base styles. However, since the grid is also defining column spacing - which is part of a site's whitespace - it is considered a base module. Persistent Grid defines the columnar structure of your site and the default behavior of columns in your project.
