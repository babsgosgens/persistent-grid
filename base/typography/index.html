Introduction

Typography at its heart is an implementation of Compass's Vertical Rhythm.

By itself, it doesn't render much output. It's main responsibility is to define default font-size and line-heigt. It's settings are used by Reset to compile the project's baseline.

Settings

Typography settings

$base-font-size
{% gist 6225110 %}
Font sizes are expressed in px but will be converted to the unit as set in $font-unit. Defaults to 16px.
$base-line-height
{% gist 6225127 %}
Line heights are expressed in px but will be converted to the unit as set in $font-unit. Defaults to 24px.
$type-default
{% gist 6225158 %}

$type-heading
{% gist 6225167 %}
An array of type properties. Defaults to $serif-regular. See: Fonts

Additional Settings

$debug-baseline
$debug-baseline: false;
This setting is useful when debugging the vertical rhythm. When set to true it will show the baseline. It is false by default. See also: @mixin debug-rhythm
