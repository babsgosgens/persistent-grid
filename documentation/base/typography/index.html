Typography

Introduction

Typography at its heart is an implementation of Compass's Vertical Rhythm.

By itself, it doesn't render much output. It's main responsibility is to define default font-size and line-heigt. It's settings are used by Reset to compile the project's baseline.

Settings

$base-font-size
Font sizes are expressed in px but will be converted to the unit as set in $font-unit. Defaults to 16px.
$base-line-height
Line heights are expressed in px but will be converted to the unit as set in $font-unit. Defaults to 24px.
$type-default
An array of type properties. Defaults to $serif-regular. See: Fonts
$type-heading
An array of type properties. Defaults to $serif-regular. See: Fonts

Baseline (Vertical Rhythm) Settings

More information on Vertical Rhythms setting scan be found here: http://compass-style.org/reference/compass/typography/vertical_rhythm/

$default-rhythm-border-style: solid !default;
$relative-font-sizing
$relative-font-sizing: true !default;
$round-to-nearest-half-line

$min-line-padding
$min-line-padding: 2px !default;
$font-unit
$font-unit: 1em !default;
$leading-unit
$leading-unit: 1em !default;

$font-unit
$font-unit: if($use-rem-for-sizing, 1rem, $font-unit) !default;
$leading-unit
$leading-unit: if($use-rem-for-sizing, 1rem, $leading-unit) !default;

Additional Settings

$debug-baseline
$debug-baseline: false;
This setting is useful when debugging the vertical rhythm. When set to true it will show the baseline. It is false by default. See also: @mixin debug-rhythm

Mixins

@mixin font
Mixin for shorthand font properties. $type is an array of four type properties: style, variant, weight, font-family. Font-sizes must be pre-converted to final unit of output.
@mixin font-style
Takes a single value from the type array and invokes the appropriate font mixin.
@mixin font-variant
Takes a single value from the type array and invokes the appropriate font mixin.
@mixin font-weight
Takes a single value from the type array and invokes the appropriate font mixin.
@mixin font-family
Takes a single value from the type array and invokes the appropriate font mixin.
@mixin font-size
Placeholder mixin for @mixin adjust-font-size-to()
@mixin line-height
Placeholder mixin for @mixin adjust-leading-to()
@mixin debug-rhythm
This mixin is useful when debugging the vertical rhythm. See also: $debug-baseline

Functions

@function px-to-relative-font-size
Placeholder function for font-sizes - converts px to the unit as set in the Vertical-Rhythm module settings
@function px-to-relative-line-height
Placeholder function for line-heights - converts px to the unit as set in the Vertical-Rhythm module settings
@function extract-type-style
Extract the style property from a type array. Eexpects an array: $style, $variant, $weight, $family;
@function extract-type-variant
Extract the variant property from a type array. Eexpects an array: $style, $variant, $weight, $family;
@function extract-type-weight
Extract the weight property from a type array. Eexpects an array: $style, $variant, $weight, $family;
@function extract-type-family
Extract the family property from a type array. Eexpects an array: $style, $variant, $weight, $family;
@function weight-number-to-string
Converts a number to a string. Expects a single number: 100 through 900.
