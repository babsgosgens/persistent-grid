Reset

Settings

$reset-box-model: true
Negates padding, margins and borders. Defaults to true.
$reset-box-sizing: true
Reset the traditional box model - borders, padding and margin will not be added to the elements width. Defaults to true.
$reset-font: true
Make sure elements inherit any font settings from their parent. Defaults to true.
$reset-text-rendering: true
Enable ligatures and more accurate kerning. Defaults to true.
$reset-html-background
Add a background color to the html tag. The background-color can be defined in the Base Styles module. Defaults to true.
$reset-baseline
Set the font-size and the line-height as set in the Typography module. Defaults to true.
$reset-ios
Prevent iOS text size adjust after orientation change, without disabling user zoom. Defaults to true.
$reset-html5
HTML5 support for legacy browsers. Defaults to true.
$reset-anchor
Consistent cross-browser behavior and improved readability. Defaults to true.
$reset-media
Consistent cross-browser behavior of img and svg elements. Defaults to true.
$reset-list-style
Render lists without list style. Defaults to true.
$reset-table
Defaults to true.
$reset-table-cell
Remove most spacing between table cells. Align all content to the left. Set font-weight to default font and weight. Re-align content vertically. Defaults to true.
$reset-input
Various fixes for browser inconsistencies among form elements. Defaults to true.
$reset-quotation
Strip quotes. Some browsers do not support the quote attribute. Defaults to false.
