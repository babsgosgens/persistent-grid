Typography

Settings

Type Settings

$base-font-size: 16px
Font sizes are expressed in px but will be converted to the unit as set in $font-unit. Defaults to 16px.
$base-line-height: 24px
Line heights are expressed in px but will be converted to the unit as set in $font-unit. Defaults to 24px (1.5*16px).
$type-default: $serif-regular
An array of type properties. Defaults to $serif-regular. See: Fonts
$type-heading: $type-default
An array of type properties. Defaults to $type-default. See: Fonts

Baseline (Vertical Rhythm) Settings

The Typography module implements Compass’s Vertical Rhythm. More information on Vertical Rhythms setting scan be found here: http://compass-style.org/reference/compass/typography/vertical_rhythm/

Additional Settings

$debug-baseline
$debug-baseline: false;
This setting is useful when debugging the vertical rhythm. When set to true it will show the baseline. It is false by default. See also: @mixin debug-rhythm

Behavior

This is the default behavior and its output. The debug style rules will only be compiled when $debug-baseline is true. The output size is 0.63Kb as shown.

Mixins

@mixin font
Mixin for shorthand font properties. $type is an array of four type properties: style, variant, weight, font-family. Font-sizes must be pre-converted to final unit of output. Takes three parameters: $type (type list1)), $font-size and $line-height.
@mixin font-style
Extracts the style value from the type list. Takes one parameter: $type (type list1)).
@mixin font-variant
Extracts the variant value from the type list. Takes one parameter: $type (type list1)).
@mixin font-weight
Extracts the weight value from the type list. Takes one parameter: $type (type list1)).
@mixin font-family
Extracts the family value from the type list. Takes one parameter: $type (type list1)).
@mixin debug-rhythm
This mixin is useful when debugging the vertical rhythm. See also: $debug-baseline

Functions

@function px-to-relative-font-size
Placeholder function for font-sizes - converts px to the unit as set in the Vertical-Rhythm module settings
@function px-to-relative-line-height
Placeholder function for line-heights - converts px to the unit as set in the Vertical-Rhythm module settings
@function extract-type-style
Extract the style property from a type array. Eexpects an array: $style, $variant, $weight, $family;
@function extract-type-variant
Extract the variant property from a type array. Eexpects an array: $style, $variant, $weight, $family;
@function extract-type-weight
Extract the weight property from a type array. Eexpects an array: $style, $variant, $weight, $family;
@function extract-type-family
Extract the family property from a type array. Eexpects an array: $style, $variant, $weight, $family;
@function weight-number-to-string
Converts a number to a string. Expects a single number: 100 through 900.
