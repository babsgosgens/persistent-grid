Introduction

Reset is a combination of different resets, most notably Compass's implementation of Eric Meyer's reset and normalize.css by Nicolas Gallagher, co-created with Jonathan Neal.

It is customized for use with Persistent Grid. Some of normalize.css has been moved to Basestyles.
