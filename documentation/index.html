Introduction

Persistent Grid Framework will provide a solid base stylesheet for all your projects. It is configurable to produce the exact output you need - no less and certainly no more. Framework consists of five modules: Type, Baseline, Reset, Grid and Base.

Type produces no output but defines your project's typography settings.

Baseline is an implementation of Compass's Vertical Rhythm plugin.

Reset is a configurable reset so that your project can start of a clean, browser consistent slate.

Grid is a configurable and highly flexible layout system.

Base defines the base styles upon which you can extend.
