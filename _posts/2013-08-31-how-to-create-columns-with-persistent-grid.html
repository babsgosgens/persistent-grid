Are you a developer that likes to keep his markup free from presentational classes? Persistent Grid provides you with different ways to create columns.

Classes or style rules

Columns can be rendered by:

  1. assigning classes (at markup level),
  2. extending the rendered behavior,
  3. using mixins.

Example: columns by assigning classes
