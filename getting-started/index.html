Getting started

Persistent Grid is a modular CSS framework built with SASS and Compass. It comes in three flavours: Grid, Base and Extended.

Grid

Grid is the most basic execution of Persistant Grid. If all you need is a good layout system to complete your project then this is your flavour of choice.

compass install persistentgrid/grid

Base

Basic provides a little extra to the grid: a set of base styles to build your project on. It includes the Grid, Reset, Type module and Base Styles modules. It provides a solid base to build your stylesheet on.

compass install persistentgrid/base

Modules

Modules1) adds some of commonly used UI elements to the mix. It's an ever growing colelction of buttons, forms, tabs and a whole lot more. And you don't have to stick with the core modules. In fact you can add or override any of the provided modules. More information is to come…

compass install persistentgrid/modules

1) Modules is still under heavy construction.

Installing Persistent Grid (pre-release)

gem install persistentgrid --pre

Creating a new Compass project:

Using all flavors:

compass create my_project -r persistentgrid --using persistentgrid

Using only the grid template:

compass create my_project -r persistentgrid --using persistentgrid/grid

Using only the base template:

compass create my_project -r persistentgrid --using persistentgrid/base

Using only the modules template:

compass create my_project -r persistentgrid --using persistentgrid/modules

Adding Persistent Grid to an existing Compass project

Edit the project configuration file and add:

require 'persistentgrid'

Then, from the command line, run:

compass install persistentgrid

Or, if you want to install one of the other template besides the project template:

compass install persistentgrid/grid

Using Persistent Grid (pre-alpha)

Make sure you have Persistent Grid installed.

For a real implementation of Persistent Grid Framework, check Persistent Grid's project pages on Github.
