Introduction

Type is the only module that is not rendering any output. It's sole responsibility is to define the project's typefaces and the default font-size and line-heigt. It's settings are then used by Reset and Baseline to compile the project's baseline.

Settings

Although technically not a setting, the Type setting file imports three type presets:

{% gist 6225036 %}

If no other typeface is used as the default type, these will be used as the project's typefaces. See Base for more information

$base-font-size
{% gist 6225110 %}
Font sizes are expressed in px but will be converted to the unit as specified in Baseline. Defaults to 16px.
$base-line-height
{% gist 6225127 %}
Like font sizes, line-heights are expressed in px and will be converted to the unit as specified in Baseline. Defaults to 24px.
